◆ am_hal_sdhc_execute_cmd()

uint32_t am_hal_sdhc_execute_cmd ( void *  pHandle,
am_hal_card_cmd_t *  pCmd,
am_hal_card_cmd_data_t *  pCmdData 
)

SDHC send command function.

Parameters
handle- handle for the interface.
pCmd- pointer to command data structure which including the command index, argument, response type, command response and command error code.
pCmdData- pointer to command related data structure, like command direction, command transfer mode, block size, block count, data buffer, etc.

This function sends a command to SD/MMC/eMMC/SDIO card and gets the response. if this command is using synchronous transfer mode, it will be blocked until the data in the buffer has been transmited or received. if this command is using asynchronous transfer mode, it will return immediately after sending the command, data transfer completion done event will be notified by registered callback function in the ISR.

Returns
status - generic or interface specific status.
