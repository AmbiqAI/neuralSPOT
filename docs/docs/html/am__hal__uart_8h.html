Initialize the UART internal RX/TX queue.

Parameters
pHandleis a pointer to a UART handle to initialize.
pui8TxBufferis the pointer to buffer with length of ui32TxBufferSize.
ui32TxBufferSizeis the length of Tx buffer.
pui8RxBufferis the pointer to buffer with length of ui32RxBufferSize.
ui32RxBufferSizeis the length of Rx buffer.

This function enables the internal UART queue (FIFO) for UART Tx/Rx. When Tx or Rx Queue are enabled, the UART data is stored internally when UART hardware FIFO is full.

Returns
HAL status code.

This function disables the UART interrupt(s) given by ui32IntMask. If multiple interrupts need to be disabled, they can be OR'ed together.

Note
This function need not be called for UART FIFO interrupts if the UART buffer service provided by am_hal_uart_buffer_configure() and am_hal_uart_interrupt_service() is already in use. Non-FIFO-related interrupts do require the use of this function.

uint32_t am_hal_uart_tx_flush ( void *  pHandle)

Wait for the UART TX to become idle.

Parameters
pHandleis the handle for the UART to operate on.

This function waits (polling) for all data in the UART TX FIFO and UART TX buffer (if configured) to be fully sent on the physical UART interface. This is not the most power-efficient way to wait for UART idle, but it can be useful in simpler applications, or where power-efficiency is less important.

Once this function returns, the UART can be safely disabled without interfering with any previous transmissions.

For a more power-efficient way to shut down the UART, check the am_hal_uart_interrupt_service() function.

Returns
AM_HAL_STATUS_SUCCESS or applicable UART errors.
