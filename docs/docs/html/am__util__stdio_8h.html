Detailed Description

A few printf-style functions for use with Ambiq products.

Functions for performing printf-style operations without dynamic memory allocation.

For further information about this module concerning its history, uses, and limitations, please see the Ambiq Micro KB article "Q&A: What does the AmbiqSuite SDK am_util_stdio_printf() function do?" at:

https://support.ambiqmicro.com/hc/en-us/articles/360040441631
