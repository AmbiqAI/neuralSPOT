void mpu_global_configure ( bool  bMPUEnable,
bool  bPrivelegedDefault,
bool  bFaultNMIProtect 
)

Sets the global configuration of the MPU.

Parameters
bMPUEnable- Enable the MPU
bPrivelegedDefault- Enable the default priveleged memory map
bFaultNMIProtect- Enable the MPU during fault handlers

This function is a wrapper for the MPU_CTRL register, which controls the global configuration of the MPU. This function can enable or disable the MPU overall with the bMPUEnable parameter, and also controls how fault handlers, NMI service routines, and priveleged-mode execution is handled by the MPU.

Setting bPrivelegedDefault will enable the default memory map for priveleged accesses. If the MPU is enabled with this value set, only priveleged code can execute from the system address map

Setting bFaultNMIProtect leaves the MPU active during the execution of NMI and Hard Fault handlers. Clearing this value will disable the MPU during these procedures.

Returns
None.
