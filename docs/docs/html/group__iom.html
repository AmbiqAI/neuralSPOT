IOM Clock Frequencies

Macro definitions for common SPI and I2C clock frequencies.

These macros may be used with the ui32ClockFrequency member of the am_hal_iom_config_t structure to set the clock frequency of the serial interfaces.

This list of frequencies is not exhaustive by any means. If your desired frequency is not in this list, simply set ui32ClockFrequency to the desired frequency (in Hz) when calling am_hal_iom_config().
